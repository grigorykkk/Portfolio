# Портфолио студента - Григорий Костин

Современное портфолио студента с ярким желто-черно-белым дизайном, созданное с использованием HTML5, CSS3, JavaScript и Bootstrap 5.

##  Описание проекта

Данный проект представляет собой полнофункциональное портфолио студента с современным дизайном в желто-черно-белой цветовой схеме. Сайт включает в себя четыре основные страницы с интерактивными элементами, анимациями и адаптивным дизайном.

###  Страницы сайта

- **Главная страница** (`index.html`) - информация о студенте, навыки с анимированными прогресс-барами и лучшие проекты
- **Страница проектов** (`pages/projects.html`) - полная коллекция работ с системой фильтрации по технологиям
- **Страница контактов** (`pages/contacts.html`) - форма обратной связи с валидацией и контактная информация
- **Учебный дневник** (`pages/diary.html`) - отслеживание прогресса обучения и текущих курсов

##  Особенности

###  Дизайн
- **Современная цветовая схема** - желто-черно-белая палитра с акцентными цветами

##  Установка и запуск

### Быстрый старт
1. Скачайте или клонируйте репозиторий
2. Откройте файл `index.html` в веб-браузере

### Локальный сервер (рекомендуется)
Для корректной работы всех функций рекомендуется запустить локальный сервер:

```bash
# Используя Python 3
python3 -m http.server 8000

# Используя Python 2
python -m SimpleHTTPServer 8000

# Используя Node.js (если установлен http-server)
npx http-server

# Используя PHP
php -S localhost:8000

# Используя Live Server (VS Code расширение)
# Установите расширение Live Server и кликните "Go Live"
```

Затем откройте браузер и перейдите по адресу `http://localhost:8000`

##  Настройка

### Персонализация данных

Для настройки под ваши данные отредактируйте следующие файлы:

1. **index.html** (строки 17, 51, 54-56):
   ```html
   <div class="student-name">Ваше Имя</div>
   <p><strong>Привет! Я студент</strong></p>
   <p>Факультет ИПТИП, группа ЭФБО-18-24</p>
   <p>Увлекаюсь веб-разработкой</p>
   ```

2. **pages/contacts.html** (строки 81, 85, 89):
   ```html
   <span>your-email@example.com</span>
   <span>GitHub: your-username</span>
   <span>Ваш Город, Страна</span>
   ```

3. **scripts/modal.js** - обновите данные проектов в объекте `projectsData` (строки 103-202)

4. **styles/styles.css** - настройте цветовую схему в переменных CSS (строки 2-57)

### Добавление новых проектов

1. **Добавьте изображения** в папку `images/`
2. **Обновите объект `projectsData`** в файле `scripts/modal.js`:
   ```javascript
   'new-project': {
       title: 'Название проекта',
       description: 'Описание проекта...',
       technologies: ['HTML', 'CSS', 'JavaScript'],
       features: ['Функция 1', 'Функция 2'],
       demo: 'https://demo-link.com',
       github: 'https://github.com/username/repo',
       images: ['project-image.jpg']
   }
   ```
3. **Добавьте карточку проекта** в `pages/projects.html`:
   ```html
   <div class="project-card" data-tech="html" onclick="openProjectModal('new-project')">
       <div class="project-title">Название проекта</div>
       <div class="project-tech">HTML, CSS</div>
   </div>
   ```

### Изменение цветовой схемы

В файле `styles/styles.css` найдите секцию `:root` и измените значения переменных:

```css
:root {
    /* Основные цвета */
    --yellow-primary: #ffc107;      /* Основной желтый */
    --yellow-secondary: #ffed4e;    /* Светло-желтый */
    --black-primary: #000000;       /* Основной черный */
    --white-primary: #ffffff;       /* Основной белый */
    
    /* Акцентные цвета */
    --accent-blue: #007bff;         /* Синий акцент */
    --accent-purple: #6f42c1;       /* Фиолетовый акцент */
    --accent-pink: #e83e8c;         /* Розовый акцент */
}
```

##  Цветовая палитра

Проект использует современную желто-черно-белую цветовую схему:

- **Основной желтый**: `#ffc107` - для акцентов и кнопок
- **Светло-желтый**: `#ffed4e` - для фонов и градиентов
- **Основной черный**: `#000000` - для текста и границ
- **Основной белый**: `#ffffff` - для фонов
- **Акцентные цвета**: синий, фиолетовый, розовый для интерактивных элементов
